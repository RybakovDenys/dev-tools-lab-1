Detailed Description

Comprehensive test cases for GameOfLife behaviors.

Covers fundamental rules (under/overpopulation, survival, birth), canonical patterns (still lifes, oscillators, spaceship), edge wrapping, bounded vs toroidal differences, input validation, and helper invariants. Pattern tests emphasize invariants (population stability, translation, bounding box non-shrink) over brittle coordinate lists for dynamic patterns.
